Agent Trading Company
======================
AI Agent가 자료 수집, 분석, 의사결정, 거래 실행, 리스크 관리를 자동으로 수행하는 시스템입니다.

agent 주요 역할
- 자료 수집 에이전트: 다양한 소스에서 실시간 데이터 수집
- 데이터 분석 에이전트: 수집된 데이터를 기반으로 시장 동향 분석
- 거래 실행 에이전트: 자동으로 거래 주문을 생성하고 실행
- 리스크 관리 에이전트: 거래 리스크를 모니터링하고 관리

python으로 작성하며 react를 사용하여 실시간 ui를 통해 agent 상태와 거래 내역 등 필요 정보 시각화
각 에이전트는 독립적으로 동작
각 에이전트간 서로 피드백 주고받으며 협력 가능
모든 에이전트는 md 파일로 리포트 제작하여 업무 수행
모든 에이전트는 리포트 내용을 읽어 다음 업무에 반영

초기 제작은 최소한의 구조만 제작.
에이전트는 확장 가능성 생각하여 제작
트레이딩은 KIS(한국투자증권) 사용하여 진행
실전 계좌와 모의 계좌로 진행, 기본값은 모의 계좌
KIS github 및 https://apiportal.koreainvestment.com/apiservice-summary 확인하여 개발 실시간이 필요하면 websocket, 아니면 api 요청으로 진행.
패키지 설치는 불필요.
각 에이전트는 LLM api 사용, 기본값은 openai api gpt-5-mini, 추후 변경 및 선택 가능하도록 변경

모든 판단(ex. 매도 손실률 제한, 금액 제한 등등)은 해당 agent가 자율적 판단.
